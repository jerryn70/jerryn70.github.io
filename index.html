Quickstart Guide

Start with a Lite list first.
If you decide that Lite isn't for you, then move upwards or downwards the chain.

  • If Lite is too harsh for your taste, downgrade to mini.
  • If Lite does not block enough, upgrade to Pro.
  • Pro still not enough? Use Xtra.
Use these free DNS resolvers to immediately start using 1Hosts:

updates every hour; supports subdomain/wildcard & CNAME blocking; unlimited queries.
